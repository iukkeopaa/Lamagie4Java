### **1. 基本概念**

- **自增运算符**：`++`
    - `i++`：先使用 `i` 的值，再将 `i` 加 1（后置自增）。
    - `++i`：先将 `i` 加 1，再使用 `i` 的值（前置自增）。
- **自减运算符**：`--`
    - `i--`：先使用 `i` 的值，再将 `i` 减 1（后置自减）。
    - `--i`：先将 `i` 减 1，再使用 `i` 的值（前置自减）。

### **2. 高频笔试题**

#### **题目 1：以下代码的输出是什么？**


```
int i = 1;
int j = i++ + ++i;
System.out.println("i=" + i + ", j=" + j);
```

**解析**：

1. `i++`：先使用 `i` 的值（1），然后将 `i` 加 1，此时 `i = 2`。
2. `++i`：先将 `i` 加 1（`i = 3`），然后使用 `i` 的值（3）。
3. `j = 1 + 3 = 4`。
4. 最终输出：`i=3, j=4`。

### **练习 1**

#### 代码：



```
int a = 10;
int b = a-- + --a + a++ + ++a;
System.out.println("a=" + a + ", b=" + b);
```

#### 解析：

1. **`a--`**：
    - 使用 `a` 的值（10），然后 `a` 减 1（`a = 9`）。
    - 表达式值：`10`。
2. **`--a`**：
    - 先将 `a` 减 1（`a = 8`），然后使用 `a` 的值（8）。
    - 表达式值：`8`。
3. **`a++`**：
    - 使用 `a` 的值（8），然后 `a` 加 1（`a = 9`）。
    - 表达式值：`8`。
4. **`++a`**：
    - 先将 `a` 加 1（`a = 10`），然后使用 `a` 的值（10）。
    - 表达式值：`10`。
5. **计算 `b`**：
    - `b = 10 + 8 + 8 + 10 = 36`。
6. **最终输出**：
    - `a=10, b=36`。

### **练习 2**

#### 代码：

```
int x = 5;
int y = x++ * --x + x-- / ++x;
System.out.println("x=" + x + ", y=" + y);
```

#### 解析：

1. **`x++`**：
    - 使用 `x` 的值（5），然后 `x` 加 1（`x = 6`）。
    - 表达式值：`5`。
2. **`--x`**：
    - 先将 `x` 减 1（`x = 5`），然后使用 `x` 的值（5）。
    - 表达式值：`5`。
3. **`x--`**：
    - 使用 `x` 的值（5），然后 `x` 减 1（`x = 4`）。
    - 表达式值：`5`。
4. **`++x`**：
    - 先将 `x` 加 1（`x = 5`），然后使用 `x` 的值（5）。
    - 表达式值：`5`。
5. **计算 `y`**：
    - `x++ * --x`：`5 * 5 = 25`。
    - `x-- / ++x`：`5 / 5 = 1`。
    - `y = 25 + 1 = 26`。
6. **最终输出**：
    - `x=5, y=26`。
